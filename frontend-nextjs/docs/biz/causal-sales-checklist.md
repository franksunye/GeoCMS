# 因果律销售作战清单：从“通话”到“成交”的必然路径 (Action Checklist)

> **原则：** 你的目标不是“说服”客户，而是通过正确的干预动作，触发客户决策的**必然结果**。

---

## 📅 第 1 阶段：建立“救援关系” (0-45秒)
**目标：** 将你的身份从“推销员”转变为“救火队员”。

*   [ ] **共情锚点**：不要急着报价格，先确认痛苦。“*漏水确实很麻烦，尤其是家里还有老人/孩子/刚装修...*”
    *   *因果逻辑*：验证痛苦 -> 提升问题的优先级 -> 合法化上门。
*   [ ] **决策身份确认**：顺口问一句。“*平时家里是您照顾吗？/ 维修的事您能定对吧？*”
    *   *因果逻辑*：排除“非决策主体”产生的无效干预。

## 🔬 第 2 阶段：通过“确定性”驱动上门 (45-120秒)
**目标：** 用“工具”代替“嘴巴”，创造“非看不可”的理由。

*   [ ] **植入“黑科技”信号**：提及 2 个以上专业名词。“*热成像仪*”、“*内窥镜*”、“*水分探测计*”。
    *   *要点*：不要讲原理，讲效果。“*我带设备过去，1分钟就能定死到底是哪儿漏。*”
    *   *因果逻辑*：消除“猜测”带来的不确定性 -> 激发好奇心与掌控欲。
*   [ ] **释放“零成本”压力**：在客户开口前主动说。“*咱们上门排查和出方案是完全免费的。*”
    *   *因果逻辑*：消除损失厌恶 -> 移除成交路径上的最大心理红灯。

## ⚡ 第 3 阶段：规避“专家陷阱”与锁定指令 (120-180秒)
**目标：** 停止教育客户，开始接管流程。

*   [ ] **降级专业深度 (核心！)**：禁止长篇大论技术方案。
    *   *错误点*：解释水的张力、基层老化、一体化施工。
    *   *修正动作*：点到为止。“*情况我明白了，这属于典型的管根渗漏，方案很多，我得现场看一眼定个最省钱的。*”
    *   *因果逻辑*：防止解释过载导致的客户决策瘫痪。
*   [ ] **指令化闭环**：不要问“行不行”，要给选项。
    *   *动作*：加微信 -> 指导发位置 -> 二选一时间。“*我先加您微信，您发个定位，一会儿4点我过去。*”
    *   *因果逻辑*：通过“惯性决策”推动结果。

---

## 🚫 绝对禁令 (Negative Interventions)

1.  **禁“过分热情”**：保持冷静、专业的工程师语调。
    *   *因果*：过分热情 = 缺单/推销；冷静专业 = 专家/救援。
2.  **禁“电话判病”**：禁止在电话里给结论。
    *   *因果*：一旦你在电话里说“肯定是XX问题”，客户就把电话挂了去网上找 DIY 方案了。
3.  **禁“长篇大论”**：通话尽量控制在 **180秒** 以内。
    *   *因果*：时间越长，转化率越低。多出的时间都在制造摩擦力。

---

## 💡 每日复盘三问
1.  我今天提“热成像仪”了吗？（增加确定性）
2.  我今天在电话里给“讲座”了吗？（避开专家陷阱）
3.  我今天主动说“免费方案”了吗？（移除损失恐惧）
